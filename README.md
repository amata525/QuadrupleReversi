##QuadrupleReversi

Android端末向けアプリとして開発された
二人対戦４色のリバーシ風ゲームです
通常のリバーシと違い、より多くの石を挟み相手のHPを0にすると勝ちです
  
GooglePlayStoreで公開しています。ぜひどうぞ。
<https://play.google.com/store/apps/details?id=com.unity3d.QuadrupleReversi>

#開発環境

ゲームエンジン：Unity4.3.4
スクリプト言語：C#  
Asset：NGUI2.7

このプロジェクトはUnityで作られているので
PC、iOSプラットフォーム向けにビルドしても動くと思います。
  
※画面比16:9向けに作られているため、iPhone5以前のiOS端末で実行する際には
NGUIのレイアウトを少々変えないと不自然に縮小されてしまいます

ライセンスフリーです。
詳しい概要は中のReadme.txtをご覧ください。
