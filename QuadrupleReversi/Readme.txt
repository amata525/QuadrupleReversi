・はじめに
　このソフトはAndroid端末向けの
　二人対戦の4色リバーシもどきです
　　　
・使い方
　GooglePlayStoreにアップしてみました
　Android端末をお持ちの方は、ぜひこちらから
  https://play.google.com/store/apps/details?id=com.unity3d.QuadrupleReversi
　
・開発目的と動機
　・Unityの基本的な使い方を覚えること
　・C#の基本的な使い方を覚えること
　・リバーシが好きなのでリバーシを題材にしたゲームが作りたかった
　・スマートフォンアプリを作ってみたかった

・開発環境
　・ゲームエンジン
　　　Unity 
　・統合開発環境
　　　Visual Studio 2013
  ・スクリプト言語
　　　C#

・詳しいゲームのルール
　・プレイヤーはグリーン、イエローの二人
　・ゲーム開始時に４つの石をどの順番で置くかを選ぶ
　・４つの石は、プレイヤーグリーンは青、赤、緑、緑の４つ
　　　　　　　　プレイヤーイエローは青、赤、黄、黄の４つ
　・プレイヤーグリーンの先攻でゲームスタート　
　・石を同じ色で挟むと、挟んだ数*列数のダメージを相手に与えて相手のターンへ
　・挟める場所が無い場合、石を空いている好きなマスに置ける。その後相手のターンへ
　・ボードの空きマスが無くなると、石を取り除き初期配置に戻る
　・どちらかのHPが0以下になるとゲーム終了
